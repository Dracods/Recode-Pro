"JavaScript é uma linguagem de script orientada a objetos, multiplataforma. É uma linguagem pequena e leve.
Dentro de um ambiente de host (por exemplo, um navegador web) o JavaScript pode ser ligado aos objetos deste ambiente
para prover um controle programático sobre eles."

-Mozilla

O Javascript interage com todos os tipos de elementos da pagina, pode trabalhar com variaveis, manipular elementos, fazer tratamento de dados tudo isso em tempo real. Pode ser usado nas paginas html de forma semelhante ao css(em linha, interno e externo), com a diferença que:

  • Se for em linha em vez de style, ele deve ser chamado através de uma ação(ou ele vai ser executado logo que a pagina for carregada).

  • Interno ele deve ficar no head dentro da tag <script> e deve ser feita uma função para ser referenciada.

  • Externo a extensão é .js e a referencia é feita diretamente na tag <script> atraves do src=""

    • Como o html carrega tudo de cima para baixo pode ocorrer do carregamento
      demorar demais ao chegar na parte dos scripts(ao chegar lá ele pausa o carregamento do html até terminar)


      Fonte: https://braziljs.org/artigos/diferencas-entre-async-e-defer/

      Para reduzirmos a lentidão ao carregar a pagina podemos usar duas propriedades:

    • Async:

      A propriedade async avisa o navegador para não pausar o carregamento da pagina ao chegar no script,
      carregando e executando os dois ao mesmo tempo.

      Fonte: https://braziljs.org/artigos/diferencas-entre-async-e-defer/

    • Defer:

      A propriedade defer também não pausa o carregamento da pagina ao chegar no script,
      porém só executa os scripts após a pagina html ester carregada totalmente

      Fonte: https://braziljs.org/artigos/diferencas-entre-async-e-defer/
